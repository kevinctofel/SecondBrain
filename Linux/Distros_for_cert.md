All distros share the same basic software architecture including:
- Linux kernel
- GNU software utilities
- A user interface
- Specific application software

## Rocky Linux

From the original CentOS team because the project joined the Red Hat Open Source and Standards team in 2014. Uses the ***dnf*** package manager.

## Ubuntu Desktop Linux

Widely used with the ***apt*** package manager. 

## openSUSE

A third option to gain experience with the ***zypper*** package manager.
